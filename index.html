HTML

  • The body element represents the visible content shown to the user.
  • Metadata elements hold information about the webpage.
  • Section elements help to organize the page to make it easier for the user to view.
  • There are many other attriibutes that can be added to a page to make the page more easily accessible to users.
  • Semantic elements can be used to indicate the role of their content and give structure to the page
